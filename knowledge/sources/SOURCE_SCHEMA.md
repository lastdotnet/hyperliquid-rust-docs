# Source Schema

Example:

```text
---
id: source.example
title: Example source title
kind: official_doc
scope: protocol
path: docs/obsidian/Example.md
url: https://example.com
captured_at: 2026-04-03
---
```

Notes:

- `path` is for local repo material
- `url` is for external primary sources
- use one or both as appropriate
- the body should explain what this source confirms or constrains
